# Authors & Acknowledgments

**Original author:** Luke Peluso
**Originally developed at:** Quality Service Company (QSC), Florence, SC

This skill was extracted from QSC's internal ServiceTitan integration work
and shared with the broader ServiceTitan / Claude Code community. The
silent-fail catalog and incident log reflect ~6 months of production
write operations against a multi-trade ST tenant (HVAC, Plumbing,
Electrical, Generators).

If you're using this skill in production and you've verified a new
ST gotcha, please file a PR — the catalog is only as good as the
data points it captures.
